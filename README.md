FE App
====

FE app consumed by the user

Setup
----

Install

`npm i`

Run on development

`npm run dev`

Run app

`npm start`

Docker commands
----

Build docker image

`npm run docker:build`

Run docker instances based on an image

`npm run docker:run`

Push docker image to repo

`npm run docker:push`

Pull docker image from repo

`npm run docker:pull`
